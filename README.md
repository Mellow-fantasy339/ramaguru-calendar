# Event Calendar Planner

A centralized repository to plan, organize, track, and review events across academic, research, innovation, and community domains.

## Purpose
This repository maintains a structured, version-controlled calendar for annual planning and execution of events, including:

- Academic schedule & departmental events  
- Institutional Innovation Council (IIC) activities
- Research/Publication timelines & workshops
- Student engagement programs & club events
- Outreach, social impact & community service events

## Features
- Annual, monthly, and category-wise calendars
- Templates for structured event planning
- CSV-based event database
- Auto-generation of `.ics` files using GitHub Actions
- Documentation folder for detailed plans
- Milestone tracker for event preparation

## How to Use
1. Add/Update events in `data/YYYY-events.csv`
2. Commit changes  
3. GitHub Action auto-generates `.ics` file  
4. Import `.ics` into Google Calendar/Outlook if needed
